WeFlow 参数说明
Base URL
本地 WeFlow fork 服务地址，只允许本机地址作为正式拉取入口。
Token Env / Token
鉴权来源；Token 留空时优先读取 Token Env 指向的环境变量。
Talkers
要拉取或回填的会话 ID；多个用逗号分隔，空值用于发现会话。
Workers
拉取媒体和消息的并发数；过高会增加磁盘、GPU/ASR/OCR 和模型摘要排队压力。
Message Limit / Max Pages
单页消息数与最多翻页数；常规 5 秒轮询建议小批量，历史回填再放大。
Lookback Seconds
初次或游标不完整时向前补看的时间窗，用于避免漏掉刚发生的消息。
Interval
后台循环拉取间隔秒数；越小越及时，但会更频繁触发文件解析和模型任务。
